CSCE 412 - Cloud Computing

CSCE 412 covers the fundamentals of cloud computing, including distributed systems, virtualization, storage, and networking in cloud environments. This page serves as a hub for course-related content, assignments, and resources.

Course Overview

About This Course

Cloud computing enables scalable, on-demand access to computing resources over the internet. Through this course, I am learning about key concepts such as cloud service models (IaaS, PaaS, SaaS), deployment models, and hands-on experience with cloud platforms. This knowledge is essential for building modern, scalable applications.

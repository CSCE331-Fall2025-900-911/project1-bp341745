Project Overview

ProxSy

ProxSy is a gesture-controlled, peer-to-peer data transfer system that enables cross-platform clipboard sharing over Bluetooth Low Energy (BLE) connection. By combining local device discovery, secure TCP communication, and computer vision-based hand gesture recognition, ProxSy allows users to initiate and confirm data transfers through intuitive hand movements. The purpose of this project is to eliminate unnecessary intermediaries in local file exchange between local devices. You can check out my project's Doxygen documentation here.

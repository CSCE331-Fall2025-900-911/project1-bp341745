CSCE 482 - Capstone

CSCE 482 is a capstone course for computer science students. This course is a collaborative project that allows students to apply their knowledge to a real-world problem.

Project Overview

In modern collaborative environments, transferring files between nearby devices remains unnecessarily complex. Users often rely on third-party cloud storage services which introduce friction, privacy concerns, compatibility limitations, and workflow interruptions. In professional environments such as meetings, classrooms, and team-based workspaces, quickly distributing data to multiple devices should be seamless and secure. However, current solutions either depend on centralized cloud services or are restricted to specific operating systems. This project introduces ProxSy, a gesture-controlled, peer-to-peer data transfer system that enables cross-platform clipboard sharing over Bluetooth Low Energy (BLE) connection. By combining local device discovery, secure TCP communication, and computer vision-based hand gesture recognition, ProxSy allows users to initiate and confirm data transfers through intuitive hand movements. The purpose of this project is to eliminate unnecessary intermediaries in local file exchange between local devices. The project documentation can be found here.
